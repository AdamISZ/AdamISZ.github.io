Main Page

This site is temporarily (for now) acting as a back-up of the former version of Joinmarket.me.

Note that the latest release of Joinmarket is always to be found here.

This site was mostly used to house my blog (see "waxwing's Blog" here); I will repopulate the site with the former articles. Please be aware that there will be some bad formatting due to difficulties with re-instantiating the site on a new platform; I'll endeavour to ensure that, at the least, the external links still work, but things like equations, some images and some special characters will be formatted wrongly. It will be readable, except the ultra-mathy stuff. At some point I may re-instantiate also the public orderbook watcher instance but this should be of little interest since it can be run trivially, locally with an install of Joinmarket (you don't need Bitcoin Core for that).
