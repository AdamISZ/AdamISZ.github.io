Donations for my work gratefully received via btcpayserver here - Tor only, and please use payjoin if you have a BIP78-supporting wallet :)).

Please note that there is a general Joinmarket donations server, also, here; so this is specifically if you want to support me.

The btcpayserver instance supports payment over Lightning as well as on-chain bitcoin (the denomination is BTC, but since it's donations, you can choose any small amount, then overpay it, if it's easier).
